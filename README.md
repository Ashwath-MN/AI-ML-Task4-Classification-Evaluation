# AI/ML Task 4 – Classification Models & Evaluation Metrics

## Project Overview
This project implements machine learning classification models to predict whether a breast tumor is malignant or benign using the Breast Cancer dataset.

## Dataset
The dataset used is the Breast Cancer dataset from the scikit-learn library. It contains 30 numerical features describing tumor characteristics.

Target classes:
- 0 → Malignant (Cancer)
- 1 → Benign (No Cancer)

## Models Implemented
1. Logistic Regression
2. Decision Tree Classifier

## Evaluation Metrics
The model performance was evaluated using:

- Confusion Matrix
- Precision
- Recall
- F1-score
- ROC Curve
- AUC Score

## Results
The Logistic Regression model achieved very strong classification performance with accuracy around 98% and ROC-AUC close to 1.0.
